# PowebPoint

Web-based presentation engine that turns HTML, CSS and JavaScript into a cinematic slide deck.

> Stop exporting static `.pptx` files. Present straight from the browser with grid overview, topic menu, fullscreen mode, keyboard navigation and per-slide animations.
